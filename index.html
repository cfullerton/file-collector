Batch File Dowloader beta

Browser based tool for dowloading a long list of urls

How It Works

Don't waste your time manually downloading files or pull your hair out trying to write a script to do it. Use this free tool to automatically download all of the files on your list. Upload a file containing a list of urls to files that you need to download. Our server finds and collects the files, then puts them into a zip file for you to download. Text or csv file goes in, zipped folder containing all of the files that you need comes out, simple as that! If you'd rather paste your list, you could do that too, either way works just fine. We've tested with lists over 50,000 files. There are no limits on the number of files with the free version during this beta period.
